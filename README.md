# JASC – Automated Waste Sorting System

This project was developed for the **Janatics Automation Skill Challenge (JASC) 2025**.
The system demonstrates an **automated waste segregation mechanism** capable of sorting different types of waste such as **paper, plastic, and metal** using industrial automation concepts.

The goal of the project was to design a **reliable and efficient sorting system** that can help improve recycling efficiency and reduce manual waste segregation.

---

## Features

* Automatic waste detection and sorting
* Sorting of **paper, plastic, and metal**
* Conveyor-based material transportation
* Integration of **Arduino and PLC control systems**
* Communication between Arduino and PLC using a **relay interface**
* Real-time sensor-based material detection
* Modular and scalable automation design

---

## System Overview

The system uses a **conveyor belt mechanism** to transport waste materials through different detection stages. Sensors detect the type of waste material and send the data to the control system.

Based on the detected material type, the system activates a **sorting mechanism** that directs the waste into the appropriate bin.

The architecture combines **embedded systems (Arduino)** with **industrial automation (PLC)** to simulate a practical automated recycling system.

## Working Principle

1. Waste items are placed on the **conveyor belt system**.
2. Sensors detect the characteristics of the material.
3. The **Arduino microcontroller processes the sensor data** and determines the material type.
4. Arduino sends control signals to the **PLC using a relay module interface**.
5. The **PLC controls the conveyor system and sorting actuators**.
6. The item is directed into the correct **waste category bin** (paper, plastic, or metal).

This hybrid system demonstrates how **embedded control systems and industrial PLC automation can work together** in real-world applications.
